# 🎵 Sargam

**Sargam** is a modern music streaming web application inspired by the clean design and seamless user experience of leading music platforms. Built using **HTML, CSS, and JavaScript**, the project focuses on delivering a premium interface with responsive layouts, elegant animations, and an intuitive browsing experience. Rather than replicating an existing platform, Sargam demonstrates modern frontend development techniques while maintaining its own unique visual identity.

The application features a dark-themed user interface enhanced with gradient accents, carefully chosen typography, and a structured layout that creates a polished and immersive experience. Users can explore featured playlists, trending tracks, popular artists, and specially curated sections such as **Non-Skippable Songs**, all presented through responsive cards and interactive components.

Sargam has been designed with simplicity and performance in mind. The project emphasizes clean code organization, reusable components, and modern CSS practices including Flexbox, Grid, transitions, hover effects, and responsive design principles. Every section is crafted to provide a smooth user experience across desktops, tablets, and mobile devices.

### Features

* Modern Spotify-inspired interface
* Responsive design for multiple screen sizes
* Premium dark theme with gradient backgrounds
* Featured playlists and albums
* Trending songs and artist sections
* Interactive hover animations and transitions
* Clean navigation and organized layout
* Optimized typography and spacing
* Built entirely with vanilla HTML, CSS, and JavaScript

### Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts

This project was developed as part of my frontend development journey to strengthen my understanding of responsive web design, UI/UX principles, layout systems, and modern styling techniques. It showcases my ability to transform a design concept into a functional and visually appealing web application while writing clean, maintainable code.
